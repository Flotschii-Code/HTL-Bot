# HTL-Bot

> A simple PlatformIO firmware program for the Wemos Lolin32 robot.

## 📝 Description

HTL-Bot is a firmware project specifically designed for controlling and adding features to the Wemos Lolin32 robot platform. It provides a simple starter structure for developers looking to program this ESP32-based microcontroller hardware using C/C++.

## ✨ Key Features

- **🤖 Wemos Lolin32 Support** — Configured specifically to target and run on Wemos Lolin32 ESP32 development boards for robotic applications.
- **⚙️ PlatformIO Integration** — Uses a standard platformio.ini configuration for managing builds, uploads, and board targets.
- **📂 Modular Project Structure** — Features dedicated directories for source files, shared headers, local libraries, and test suites.

## 🎯 Use Cases

- Developing custom robot behaviors and controls on a Wemos Lolin32 ESP32 development board.
- Setting up a clean, PlatformIO-based C/C++ environment for hardware prototyping and firmware compilation.
